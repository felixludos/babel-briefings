Babel Briefings

A News Headline Dataset from Around the World

Presented at NAACL 2024 by Felix Leeb

Dataset Overview

This dataset contains 4,719,199 news headlines across 30 different languages collected between 8 August 2020 and 29 November 2021. The headlines were collected using the News API by collecting the top headlines (usually about 30-70 articles) separately for each combination of the 54 locations x 7 categories almost every day. Note, that the same article may occur more than once across different locations, categories, or dates (which is recorded in the `instances` property), so in total 7,419,089 instances were collected.

Get the Dataset

The dataset is available for download on HuggingFace and Kaggle, with the code available on GitHub. The associated (short) paper is published in the NAACL 2024 proceedings and available on arXiv.
